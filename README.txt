# Securities Streamlit App

אפליקציית Streamlit לניתוח רכישות ומכירות של ניירות ערך מקובץ XLSX.

## יכולות
- העלאת קובץ XLSX
- קריאה של לשונית "הכל"
- חיפוש לפי שם נייר או מספר נייר
- סינון לפי ענף וסוג נייר
- סיכום לפי סוג משקיע
- פירוק לפי חיתוכים
- פיצול משקיע ישראלי ליחיד/תאגיד
- איחוד תושב חוץ לקבוצת "משקיע חוץ"
- גרפים בסיסיים
- ייצוא CSV

## התקנה
```bash
pip install -r requirements.txt
streamlit run app.py
```

## פריסה ל-Streamlit Cloud
ראה DEPLOY_STREAMLIT_CLOUD.txt
